## HaulCalc v1.7.0

Desktop release for **Reporting & Localization Q2 2026** (parent DAS-283): IDR display currency, richer PDF/JSON exports, and shell polish.

### Features

- **Currency settings (DAS-292)** -- USD/IDR display currency and configurable USD-to-IDR rate in Settings (#298)
- **Currency formatting (DAS-293)** -- Thousand separators and locale-aware money display app-wide (#302)
- **Dynamic chart labels (DAS-294)** -- Chart legends and economics tables use active currency units (#305)
- **PDF chart images (DAS-289)** -- Optional opex stacked bar and LCCA cumulative charts embedded in design report PDF (#306)
- **PDF sensitivity section (DAS-290)** -- Optional sensitivity analysis table and chart in PDF/JSON; snapshot v4 in project files (#307)
- **PDF compare projects (DAS-291)** -- Optional cross-project comparison tables in PDF/JSON from Compare tab (#308)
